# A. A<=B问题

## 题目描述

上回书说道，Askalana梦见她考上了一所大专，课上，老师在狂吹某编程网站，他讲了一个有趣的小问题。构造一个长度为 $n$ 的排列 $a$ ，使得满足 $\exists (i, j) \in [1, n]$ 有 $i < j$ 且 $\forall (i, j) \in [1, n]$ 且 $i < j$ 有 $a_j \leq a_i$ 。  


## 输入描述

第一行一个正整数 $n(1\leq n \leq 2 \times 10^5)$ ，表示所求数列长度。

## 输出描述

 $n$ 个正整数，表示所求的数列。

## 样例

### 样例 1
**输入**:
```
1
```

**输出**:
```
-1
```
