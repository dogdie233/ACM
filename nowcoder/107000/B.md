# B. 小红开灯（二）

## 题目描述

有 $n$ 盏灯排成一排，初始为“开启”或者“关闭”的状态。  

小红每次操作可以关闭任意一盏灯，她希望不存在两盏相邻的灯同时开启，请你帮小红计算至少需要关掉多少盏灯。  


## 输入描述

第一行输入一个正整数 $n$ ，代表灯的数量。  
第二行输入一个长度为 $n$ 的01串，代表每盏灯的初始状态。'0'代表关闭，'1'代表开启。  
$1\leqq n \leqq 10^5$   


## 输出描述

一个整数，代表小红最少需要关闭的灯的数量。  


## 样例

### 样例 1
**输入**:
```
4
0110
```

**输出**:
```
1
```

**说明**:  

小红关闭第二盏灯即可。
